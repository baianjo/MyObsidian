# 🧠 存储系统 (1)：层次结构与分类

> **💡 导语 & 备考策略**

## 一、 存储器的层次结构 (核心考点)

计算机的存储系统像一个金字塔，**越往上：速度越快、容量越小、造价越高**。

### 1. 金字塔结构与访问权限
*   **寄存器 (Registers)**：在CPU内部，最快，容量极小（几十个）。
*   **高速缓存 (Cache)**：在CPU内部/边缘，分为L1/L2/L3。速度极快，容量较小。
*   **主存 (Main Memory / 内存)**：速度较快，容量适中（如8GB/16GB）。
*   **辅存 (Auxiliary Storage / 外存)**：如磁盘、SSD。速度慢，容量极大（如1TB）。
    *   *注：有时辅存指机箱内的硬盘，外存指U盘/光盘，做题时视语境等同处理即可。*

🚨 **【绝对红线：CPU的访问权限】**
*   **CPU 可以直接读写**：寄存器、Cache、主存。
*   **CPU 绝对不能直接访问**：辅存（外存）。辅存的数据**必须**先调入主存，才能被CPU处理！（例如：开机慢、启动微信要等几秒，就是在把辅存数据搬进主存）。

### 2. 两个关键的数据交换层 (⭐⭐⭐ 必考对比)

这是本节最容易出选择题的地方，用表格对比记忆：

| 层次对比 | Cache - 主存层次 | 主存 - 辅存层次 |
| :--- | :--- | :--- |
| **解决的核心矛盾** | 解决 **CPU与主存速度不匹配** 的问题 | 解决 **主存容量不够** 的问题 |
| **实现技术** | **纯硬件** 自动完成 | 硬件 + **操作系统** 共同完成 |
| **构成的系统** | 硬件级存储系统 | **虚拟存储系统** (Virtual Memory) |
| **对谁透明 (考点)** | 对 **所有程序员** (应用+系统) 都透明 | 对 **应用程序员** 透明，对 **系统程序员(OS)** 不透明 |

> *🗣️ 人话翻译“透明”：透明就是“看不见，不需要你管”。Cache的数据搬运完全由硬件包办，写代码的人根本不用操心；但主存和辅存的页面置换，写操作系统的程序员（系统程序员）是需要写代码去调度的。*

## 二、 存储器的分类 (防丢分名片库)

考试常常给出一个具体的存储器（如快表、U盘、DRAM），问你它属于哪种分类。必须精准对号入座。

### 1. 按“存取方式”分类 (最重要！)
*   **随机存取存储器 (RAM)**：指读写任何一个位置的时间都是一样的，与物理位置无关。
    *   *示例：主存（内存条）。*
*   **顺序存取存储器 (SAM)**：读写时间取决于数据所在的物理位置（只能顺着往下找）。
    *   *示例：磁带。*
*   **直接存取存储器 (DAM)**：介于RAM和SAM之间。先“随机”跳到某个区域，再“顺序”找。
    *   *示例：机械硬盘（磁盘）。先移动磁头（随机），再等盘片转过来（顺序）。*
    *   **⚠️ 考研陷阱**：SAM 和 DAM 统称为 **串行访问存储器**。
*   **相联存储器 (CAM / 按内容访问)**：**不按地址！按内容检索！** 告诉它你想找什么特征的数据，它直接告诉你位置。
    *   *示例：**快表 (TLB)**。（记住这个例子，逢考必出）*

### 2. 按“信息可保存性”分类 (断电后数据还在吗？)
*   **易失性存储器**：断电即清空。
    *   *示例：主存 (RAM)、Cache。*
*   **非易失性存储器**：断电数据保留。
    *   *示例：辅存（磁盘、固态硬盘SSD）、光盘、ROM（如主板上的BIOS）。*

### 3. 按“读出是否破坏数据”分类 (重点，衔接下一章)
*   **破坏性读出**：读一次，原数据就没了，**必须紧接着进行重写（恢复）**。
    *   *示例：**DRAM**（动态RAM，常作主存）。*
*   **非破坏性读出**：读完原数据还在。
    *   *示例：SRAM（静态RAM，常作Cache）、磁盘、光盘等绝大多数存储器。*

### 4. 按“信息可更改性”分类
*   **读写存储器**：可读可写 (磁盘、内存、Cache)。
*   **只读存储器 (ROM)**：通常只能读不能写 (CD-ROM, 蓝光光盘, 存BIOS的芯片)。*注：现代ROM其实可以通过特殊手段擦写，但大类上仍归为ROM。*

## 三、 存储器的性能指标 (计算题基石)

### 1. 存储容量
*   **公式**：`存储容量 = 存储字数 × 存储字长`
*   **关联回顾**：MAR的位数决定了存储字数（地址空间），MDR的位数决定了存储字长。

### 2. 单位成本
*   **公式**：`单位成本 = 总价格 / 总比特数(Bits)`
*   *注意单位统一，算的是每个 **比特(bit)** 的钱。*

### 3. 数据传输率 (主存带宽)
*   表示每秒从主存进出信息的最大数量。
*   **公式**：`数据传输率 (带宽) = 数据的宽度(存储字长) / 存取周期`

🚨 **【🎯 终极易错考点：存取时间 vs 存取周期】**
*   **存取时间 ($T_a$)**：单纯完成一次读/写操作需要的时间。
*   **存取周期 ($T_m$)**：进行**连续两次**独立读/写操作之间，所需的**最短时间间隔**。
*   **核心关系**：`存取周期 = 存取时间 + 恢复时间`
*   **⚠️ 考研陷阱**：题目如果问“连续读写”，用的时间参数**一定是存取周期**！因为每次读写完，机器需要喘口气（恢复时间），不能无缝衔接。**存取周期 永远大于 存取时间。**
