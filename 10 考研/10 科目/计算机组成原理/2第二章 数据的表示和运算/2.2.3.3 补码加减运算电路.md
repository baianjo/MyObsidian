# 补码加减运算电路与标志位 (ALU核心)

> [!abstract] 核心考点直击
> 考研408或985自命题中，**加法器电路原理**及**四个标志位（OF, SF, ZF, CF）的生成逻辑**是必考选择题，甚至大题。
> **一句话总结**：计算机底层**只有加法器**，减法是通过“按位取反，末位加1”转成加法的；且**底层硬件根本不区分有符号还是无符号**，全靠一套电路算，算完后丢给不同的“标志位”去解释。

---

## 一、 补码加减运算电路图解 (说人话版)

硬件怎么既算加法又算减法？靠一个核心控制信号：`Sub` (Subtraction)。
假设输入是 $n$ 位的 $A$ 和 $B$。

*   **当做加法时 (`Sub = 0`)：**
    *   多路选择器放行 $B$ 的原值。
    *   加法器最低位进位 $C_{in} = Sub = 0$。
    *   **电路实际在算**：$A + B + 0 \Rightarrow A + B$
*   **当做减法时 (`Sub = 1`)：**
    *   多路选择器把 $B$ 接入**非门**，放行 $\overline{B}$ (按位取反)。
    *   加法器最低位进位 $C_{in} = Sub = 1$。
    *   **电路实际在算**：$A + \overline{B} + 1 \Rightarrow A - B$ (这完美契合了补码减法公式：减去一个数等于加上它的相反数的补码，即按位取反末位加1)。

> [!example] 快速推演示例 (四位补码)
> $A = 3 \rightarrow 0011$
> $B = 4 \rightarrow 0100$
> *   **算 A+B**：`Sub=0`。加法器输入 0011 + 0100 + 0 = 0111 (7)，正确。
> *   **算 A-B**：`Sub=1`。$B$ 取反变 1011。加法器输入 0011 + 1011 + 1 = 1111 (翻译为十进制就是 -1)，正确。

## 二、 👑 必考：四大标志位 (重点默写区)

> [!warning] 丢分预警
> 必须严格区分哪个标志位是给**有符号数（补码）**看的，哪个是给**无符号数**看的！底层生成逻辑（用什么逻辑门）是高频考点。

### 1. OF (Overflow Flag) - 溢出标志
*   **管谁**：专管**有符号数 (带符号数)** 加减运算。
*   **含义**：`OF = 1` 发生溢出；`OF = 0` 未溢出。
*   **硬件生成逻辑**：$OF = C_{out} \oplus C_{n-1}$ 
    *   *(最高位产生的进位 $\oplus$ 次高位产生的进位)*。
    *   只要这两个进位不一样，异或结果为1，就是溢出。

### 2. SF (Sign Flag) - 符号标志
*   **管谁**：专管**有符号数 (带符号数)**。
*   **含义**：`SF = 1` 结果为负；`SF = 0` 结果为非负(正或零)。
*   **硬件生成逻辑**：$SF = \text{最高位的本位和}$。
    *   其实就是直接把结果的最高位（符号位）抄过来。

### 3. ZF (Zero Flag) - 零标志
*   **管谁**：都管（有符号、无符号通用）。
*   **含义**：`ZF = 1` 结果**是0**；`ZF = 0` 结果**非0**。*(注意这里是反直觉的，结果是0，标志位反而是1)*。
*   **硬件生成逻辑**：对加法器输出的 $n$ 个比特进行**或非 (NOR)** 运算。
    *   全0相或为0，再取非得1。只要有一位是1，或非结果就是0。

### 4. CF (Carry Flag) - 进/借位标志 (🔥最易错考点)
*   **管谁**：专管**无符号数**加减运算。
*   **含义**：`CF = 1` 发生溢出（加法进位或减法借位）；`CF = 0` 未溢出。
*   **硬件生成逻辑**：$CF = C_{out} \oplus Sub$
    *   *(最高位输出进位 $C_{out}$ $\oplus$ 最低位初始进位 $C_{in}$)*。
    *   **为什么这么算？（彻底理解它，别死记）：**
        *   **做加法 (`Sub=0`)**：$CF = C_{out} \oplus 0 = C_{out}$。最高位如果进位了($C_{out}=1$)，说明无符号数装不下了，溢出，此时 $CF=1$。
        *   **做减法 (`Sub=1`)**：$CF = C_{out} \oplus 1 = \overline{C_{out}}$ (即取反)。减法中，最高位**没有进位 ($C_{out}=0$)**，说明你不够减，向更高位**借位**了！此时 $C_{out}$ 是0，异或1之后 $CF=1$，提示溢出！

## 三、 考场防坑终极真理 (True/False 判别)

1.  **“无符号数和有符号数需要两套不同的加减法电路”** $\rightarrow$ **错！** 底层运算逻辑一模一样，共用同一套电路。只是算完以后，程序员/编译器看不同的标志位而已（有符号看OF，无符号看CF）。
2.  **“无论加减，CF标志都等于最高位进位 $C_{out}$”** $\rightarrow$ **错！** 加法时 $CF = C_{out}$，减法时 $CF = \overline{C_{out}}$。
3.  **“如果 ZF=1，说明结果溢出了”** $\rightarrow$ **错！** ZF只表示结果是不是全0。
4.  **“两个正数相加，如果最高位产生了进位，OF一定为1”** $\rightarrow$ **不一定！** OF是 $C_{out} \oplus C_{n-1}$，要结合次高位进位一起看。无脑看最高位进位的是CF（判定无符号数用的）。
