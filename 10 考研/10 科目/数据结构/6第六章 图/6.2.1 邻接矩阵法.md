> [!tip] 985上岸策略：拿分必看
> 邻接矩阵是图存储中最基础、最直观的结构。
> **得分点**：度数计算（特别是这类有向图的出/入度）、空间复杂度（$O(|V|^2)$）、**稠密图**的首选。
> **防坑点**：无向图的对称性、网（带权图）的$\infty$处理。
> **高阶考点（压轴小题）**：$A^n$ 的含义（路径数目）。

### 一、 核心定义与代码实现

用两个数组表示图：
1.  **顶点表（一维数组）**：存顶点信息（如 `char Vex[MAX]`）。
2.  **邻接矩阵（二维数组）**：存边/弧的信息（如 `int Edge[MAX][MAX]`）。

#### 1. 逻辑结构可视化

| 类型 | 矩阵特点 | 判定规则 | 备注 |
| :--- | :--- | :--- | :--- |
| **无向图** | **对称矩阵** | $A[i][j] = 1 \iff (v_i, v_j) \in E$ | 对角线为0（无自环）；一条边对应2个"1" |
| **有向图** | 非对称 | $A[i][j] = 1 \iff <v_i, v_j> \in E$ | 行表示**出**，列表示**入** |
| **网(带权)** | 存权值 | $A[i][j] = w_{ij}$ 或 $\infty$ | 不相邻填 $\infty$ (通常用 `INT_MAX` 宏定义) |

#### 2. 代码模板 (C语言风格)

```c
#define MaxVertexNum 100
#define INFINITY 65535 // 表示无穷大

typedef char VertexType; // 顶点数据类型
typedef int EdgeType;    // 边权值类型 (不带权图可用 bool/int 0,1)

typedef struct {
    VertexType Vex[MaxVertexNum];      // 顶点表
    EdgeType Edge[MaxVertexNum][MaxVertexNum]; // 邻接矩阵
    int vexnum, arcnum;                // 当前顶点数和边数
} MGraph;
```

> [!NOTE] 空间优化技巧
> 若只需存 0/1，`EdgeType` 可用 `bool` 或枚举，甚至位运算压缩，但这通常是工程考量，**考研笔试写 `int` 绝不出错**。

### 二、 度数计算（必考，秒杀题）

给定矩阵，如何快速求某个顶点 $v_i$ 的度？

#### 1. 无向图
- **度 ($TD$)** = 第 $i$ 行（或第 $i$ 列）非零元素的个数。
- **公式**：$TD(v_i) = \sum_{j=0}^{n-1} A[i][j]$

#### 2. 有向图 (注意区分行列)
记忆口诀：**行出列入**。

- **出度 ($OD$)** = 第 $i$ **行** 非零元素之和。
- **入度 ($ID$)** = 第 $i$ **列** 非零元素之和。
- **总度 ($TD$)** = $OD(v_i) + ID(v_i)$。

### 三、 性能分析与适用场景

#### 1. 空间复杂度
- **公式**：$O(|V|^2)$  (即 $O(n^2)$)。
- **分析**：只与顶点数有关，与边数无关。
- **结论**：
    - **适合：稠密图** (边很多，$E \approx |V|^2$)。
    - **不适合：稀疏图** (浪费大量空间存0)。

#### 2. 存储优化
- 对于**无向图**，矩阵是对称的，可以只存储上三角或下三角（压缩存储），将空间减半。[[3.4 特殊矩阵的压缩存储]]

### 四、 高阶性质：矩阵幂 $A^n$ 的含义（易丢分点）

这是985考研中可能出现的**选择题难点**或**算法填空题**背景。

> [!important] $A^n$ 的物理意义
> 设 $A$ 是图 $G$ 的邻接矩阵（元素为0或1），则 $A^n$（矩阵的 $n$ 次方）中：
> 元素 **$A^n[i][j]$** 的值 = 从顶点 $v_i$ 到顶点 $v_j$ 之间**长度恰好为 $n$ 的路径数目**。

#### 示例推演
假设 $A$ 是某图的邻接矩阵：
1.  **$A^1[i][j]=1$**：$i \to j$ 有 1 条长度为 1 的边（直连）。
2.  **$A^2[i][j]=k$**：$i \to j$ 有 $k$ 条长度为 2 的路径。
    *   *原理*：矩阵乘法 $C_{ij} = \sum a_{ik} \cdot a_{kj}$。
    *   *解释*：即是否存在中间点 $k$，使得 $i \to k$ (1) 且 $k \to j$ (1) 同时成立。
3.  **$A^3[i][j]=m$**：$i \to j$ 有 $m$ 条长度为 3 的路径。

**应用场景**：
- 题目问：求顶点A到顶点D长度为3的路径有多少条？
- 解法：计算 $A^3$，找对应位置的元素值。

### 五、 极简复习总结 (Cheat Sheet)

| 考查维度      | 关键记忆点                        |
| :-------- | :--------------------------- |
| **存储结构**  | 数组 (随机存取方便)。                 |
| **空间复杂度** | $O(V^2)$，极其浪费空间，仅适合**稠密图**。  |
| **无向图度数** | 某行的和。                        |
| **有向图度数** | **出度=行和**，**入度=列和**。         |
| **网(带权)** | 不相邻填 $\infty$，自己到自己填 0。      |
| **特殊性质**  | $A^n[i][j]$ = 长度为 $n$ 的路径条数。 |
