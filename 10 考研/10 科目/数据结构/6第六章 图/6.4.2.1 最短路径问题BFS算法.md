## 1. 最短路径考点总览

考研中图的最短路径问题主要分为两类，对应三种核心算法。请务必背诵下表适用场景：

| 问题类型 | 描述 | 适用算法 | 适用图类型 | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **单源最短路径** | 求从**一个**源点到其他所有顶点的最短路径 | **BFS 算法** | **无权图** (或权值全为1) | 本节重点 |
| | | **Dijkstra 算法** | **带权图**、无权图 | 不适用负权边 |
| **各顶点间最短路径** | 求**每一对**顶点之间的最短路径 | **Floyd 算法** | **带权图**、无权图 | 允许负权边，不允许负权回路 |

> [!INFO] **无权图的本质**
> 无权图可视为一种特殊的带权图，即每一条边的权值都为 **1**。

## 2. BFS 求解单源最短路径 (无权图)

### 2.1 核心思想与数据结构
在标准 BFS (广度优先遍历) 的基础上，增加两个辅助数组来记录信息：

1.  **`d[]` (Distance)**: 记录从源点到当前顶点的最短路径长度。
2.  **`path[]` (Predecessor)**: 记录最短路径上该顶点的**直接前驱**（即从哪个点过来的），用于还原路径。

### 2.2 算法流程 (功利化记忆版)

1.  **初始化**:
    *   `d[]` 全部设为 $\infty$ (无穷大)。
    *   `path[]` 全部设为 -1。
    *   `d[源点] = 0`，`visited[源点] = true`，源点入队。
2.  **迭代 (标准BFS框架)**:
    *   队头元素 $u$ 出队。
    *   遍历 $u$ 的所有邻接点 $w$。
    *   **关键修改**: 若 $w$ 未被访问 (`!visited[w]`)：
        *   `d[w] = d[u] + 1` (距离累计)
        *   `path[w] = u` (记录前驱)
        *   `visited[w] = true`
        *   $w$ 入队

### 2.3 伪代码实现

```cpp
// 假设图以邻接表存储
void BFS_Min_Distance(Graph G, int u) {
    // 1. 初始化
    for(i=0; i<G.vexnum; ++i){
        d[i] = INFINITY; // 距离初始化为无穷
        path[i] = -1;    // 前驱初始化为-1
    }
    d[u] = 0;
    visited[u] = TRUE;
    EnQueue(Q, u);
    
    // 2. BFS 主循环
    while(!IsEmpty(Q)){
        DeQueue(Q, u); // 队头元素 u 出队
        
        // 检查 u 的所有邻接点 w
        for(w = FirstNeighbor(G, u); w >= 0; w = NextNeighbor(G, u, w)){
            if(!visited[w]){ // 只有未访问过的才处理，保证最短（因为BFS按层遍历）
                d[w] = d[u] + 1; // ==核心：更新距离==
                path[w] = u;     // ==核心：记录前驱==
                visited[w] = TRUE;
                EnQueue(Q, w);
            }
        }
    }
}
```

## 3. 实例推演与路径还原

假设以顶点 **2** 为源点进行 BFS：

**过程可视化 (分层结构):**
*   **第 0 层 (d=0)**: 2
*   **第 1 层 (d=1)**: 1, 6 (直接与2相邻，`path`均为2)
*   **第 2 层 (d=2)**: 5, 3, 7 (由1或6发现，`path`为1或6)
*   **第 3 层 (d=3)**: 4, 8 (由3或7发现，`path`为3或7)

**如何利用 `path[]` 还原路径？**
如果要找 **2 到 8** 的最短路径：
1.  查 `path[8]` -> 得到 **7** (说明是 ... $\to 7 \to 8$)
2.  查 `path[7]` -> 得到 **6** (说明是 ... $\to 6 \to 7 \to 8$)
3.  查 `path[6]` -> 得到 **2** (说明是 $2 \to 6 \to 7 \to 8$)
4.  2 是源点，结束。
5.  **逆向输出**: 路径为 $2 \to 6 \to 7 \to 8$，长度为 `d[8] = 3`。

## 4. 考研必记结论

> [!important] **BFS 生成树性质**
> 对无权图进行广度优先遍历，所得到的**广度优先生成树**，是以源点为根的所有生成树中，**高度（深度）最小**的那一棵。
> *解释：BFS 是按“层”向外扩展的，每一层就是距离增加 1，因此保证了到达每个节点的路径都是边数最少的。*
