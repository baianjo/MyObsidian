> 本节是**选择题**重灾区。重点不在于死记定义，而在于**边界条件**（如：顶点集不能为空、连通的最少/最多边数）。
> 目标：看到题目中的术语（如“强连通”、“生成树”）能立刻反应出其背后的数学性质（边数关系、度数关系）。

## 一、 图的定义与基本术语

### 1. 定义核心 ($G=(V, E)$)
- **$V$ (Vertex)**: 顶点集。**注意：$V$ 一定非空**（图不能像树那样为空树）。
- **$E$ (Edge)**: 边集。$E$ 可以为空（只有点没有边）。
- **简单图 vs 多重图**:
    - **考研默认讨论简单图**：① 不存在重复边；② 不存在自环（自己连自己）。

### 2. 无向图 vs 有向图 (度数关系)

| 特性                 | 无向图 (Undirected)  | 有向图 (Directed)                  |                              |                          |
| :----------------- | :---------------- | :------------------------------ | ---------------------------- | ------------------------ |
| **边表示**            | $(v, w)$ 圆括号，无序   | $<v, w>$ 尖括号，有序 (v为弧尾，w为弧头)     |                              |                          |
| **度数 (Degree)**    | $TD(v)$           | $TD(v) = ID(v) + OD(v)$ (入度+出度) |                              |                          |
| **握手定理**<br>(必考公式) | $\sum TD(v) = 2E$ | $$所有顶点度数之和 = 边数 \times 2$$      | $\sum ID(v) = \sum OD(v) =E$ | $$<br>入度之和 = 出度之和 = 边数$$ |

## 二、 连通性与连通分量 (高频考点)

### 1. 连通概念辨析
- **路径 (Path)**: 顶点序列。
    - **简单路径**: 顶点不重复。
    - **距离**: 最短路径长度（无路径则为 $\infty$）。
- **回路/环 (Cycle)**: 起点=终点。
    - **简单回路**: 除起点终点外，其余顶点不重复。

### 2. 连通图 vs 强连通图
> [!failure] 陷阱提示
> 看到 **"强"** 字，特指 **有向图**。

| 类型 | 连通定义 | 子图概念 | 极小连通子图 |
| :--- | :--- | :--- | :--- |
| **无向图** | 任意两点 $v, w$ 有路径 | **连通分量** (极大连通子图) | **生成树** (包含所有顶点) |
| **有向图** | 任意两点 $v \leftrightarrow w$ 双向可达 | **强连通分量** (极大强连通子图) | 只有入度为0的**有向树** (不常用) |

### 3. 边数极值问题 (背诵！)

此处为**985上岸**必须拿下的死分，考研数学思维体现：

#### A. 连通图 (无向)
1. **最少边数** (保证连通): **$n-1$** (即生成树)。
2. **非连通图的最多边数**: **$C_{n-1}^2$**。
    *   *原理*: 让 $n-1$ 个点构成完全图，剩下一个点孤立。再加一条边必然连通。

#### B. 强连通图 (有向)
1. **最少边数** (保证强连通): **$n$** (构成一个大环)。

## 三、 特殊形态的图 (性质速查)

### 1. 完全图 (Complete Graph)
任意两个顶点之间都存在边/弧。

- **无向完全图**: 边数 = **$C_n^2 = \frac{n(n-1)}{2}$**
- **有向完全图**: 边数 = **$2C_n^2 = n(n-1)$**

### 2. 树与森林
- **树**: $n$ 个顶点的**连通**图，且**无回路**。
    - 边数恒为 **$n-1$**。
    - *推论*: 边数 $> n-1$ 且连通 $\rightarrow$ 一定有环。
- **森林**: $m$ 棵树的集合。

### 3. 生成树与生成森林
- **生成子图**: 包含原图**所有顶点**。
- **生成树**: 包含所有顶点的**极小连通子图** ($V$ 不变，边砍到只剩 $n-1$ 条且连通)。
    - **应用**: 最小生成树 (Prim/Kruskal) 用于低成本构建网络。

### 4. 网 (Network)
- 边带权值 (Weight)。
- **带权路径长度**: 路径上所有边的权值之和。

## 四、 易混淆概念自测 (One-Check)

- [ ] **Q1**: 两个图顶点集相同，边集为空，它们是同一个图吗？
    - *A*: 是。图可以是 $E=\emptyset$。
- [ ] **Q2**: 强连通图一定是完全图吗？
    - *A*: 错。强连通只需构成环即可 ($n$ 条边)，完全图需要 $n(n-1)$ 条边。
- [ ] **Q3**: 极大连通子图是唯一的吗？
    - *A*: 是（对于给定的连通分量）。**注意**：不要把“极大连通子图”（连通分量，确定的）和“极小连通子图”（生成树，不唯一）搞混。
    - **极大**: 加一条边就不属于该类型（变非连通或变其他性质）。
    - **极小**: 减一条边就不属于该类型（变非连通）。

> [!abstract] **Obsidian 记忆卡片 (Cloze)**
> 1. $n$ 个顶点的无向图，若为非连通图，最多有 **$C_{n-1}^2$** 条边。
> 2. $n$ 个顶点的有向图，若为强连通图，最少有 **$n$** 条边。
> 3. 无向图所有顶点度数之和等于边数的 **2** 倍。
> 4. 图的顶点集 $V$ 必须是 **非空** 的。
