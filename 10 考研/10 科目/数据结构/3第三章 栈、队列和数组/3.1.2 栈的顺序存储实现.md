### 1. 核心定义 (代码模板)

顺序栈利用**静态数组**实现，连续存储。考研中重点考察`top`指针的两种不同含义。

```c
#define MaxSize 10  // 定义栈中元素的最大个数

typedef struct {
    ElemType data[MaxSize]; // 静态数组存放栈中元素
    int top;                // 栈顶指针
} SqStack;
```

---

### 2. 核心考点：Top 指针的两种逻辑 (极易混淆)

**考研命题陷阱**：题目未明确说明时，需根据代码逻辑或初始值判断属于哪种情况。

> [!danger] 核心差异图解
> **方案 A (常规):** 指针指向**当前**已有元素的栈顶。
> **方案 B (教材/变体):** 指针指向**下一个**可以插入的空位。

| 比较维度 | **方案 A (指向栈顶元素)** | **方案 B (指向下一个空位)** |
| :--- | :--- | :--- |
| **初始化** | `top = -1` | `top = 0` |
| **判空条件** | `top == -1` | `top == 0` |
| **判满条件** | `top == MaxSize - 1` | `top == MaxSize` |
| **入栈 (Push)** | **先加后存**<br>`S.data[++S.top] = x;` | **先存后加**<br>`S.data[S.top++] = x;` |
| **出栈 (Pop)** | **先取后减**<br>`x = S.data[S.top--];` | **先减后取**<br>`x = S.data[--S.top];` |
| **读栈顶** | `return S.data[S.top];` | `return S.data[S.top - 1];` |

### 3. 标准操作代码 (以方案 A 为例)

此类代码建议**熟读背诵**，考试时直接默写。注意 `&` 引用符号的使用。

#### 3.1 初始化
```c
void InitStack(SqStack &S) {
    S.top = -1; // 指向 -1 表示空
}
```

#### 3.2 进栈 (Push)
```c
bool Push(SqStack &S, ElemType x) {
    if (S.top == MaxSize - 1) // 1. 判满
        return false;
    
    // 2. 先指针加1，再存入数据 (等价于 S.top++; S.data[S.top]=x;)
    S.data[++S.top] = x; 
    return true;
}
```

#### 3.3 出栈 (Pop)
```c
bool Pop(SqStack &S, ElemType &x) { // 注意 x 需要带回数据，用引用
    if (S.top == -1) // 1. 判空
        return false;
    
    // 2. 先取出数据，再指针减1 (等价于 x=S.data[S.top]; S.top--;)
    x = S.data[S.top--]; 
    return true;
}
```

### 4. 进阶考点：共享栈 (Shared Stack)

为了提高内存利用率，将两个栈底分别设置在数组的两端，向中间延伸。

*   **物理结构**：一个数组 `data[MaxSize]`。
*   **逻辑结构**：
    *   **0号栈**：`top0` 从 `-1` 开始，向后增长 (`++top0`)。
    *   **1号栈**：`top1` 从 `MaxSize` 开始，向前增长 (`--top1`)。

> [!summary] 关键公式 (必考)
> *   **栈满条件**：`top0 + 1 == top1` (即两指针相邻)
> *   **0号栈空**：`top0 == -1`
> *   **1号栈空**：`top1 == MaxSize`

### 5. 秒杀复习 Tip (不丢分策略)

1.  **审题第一**：看到题目先看 `top` 初始值。是 `-1` 还是 `0`？这一步错了，后面全错。
2.  **运算顺序**：
    *   `top = -1` 时：入栈是“**加加**top” (`++top`)。
    *   `top = 0` 时：入栈是“top**加加**” (`top++`)。
3.  **引用符号**：手写代码时，`Init`、`Push`、`Pop` 的栈 `S` 都要加 `&` (C++语法)，`Pop` 的返回值 `x` 也要加 `&`。
4.  **复杂度**：所有基本操作均为 **O(1)**。
