## 一、 核心定义与操作矩阵

双端队列（Deque）本质是**插入/删除操作受限**的线性表。
> [!NOTE] 核心考点
> 考试不考定义背诵，只考**给定输入序列（通常为1,2,3,4），判断输出序列合法性**。

| 类型 | 允许插入端 | 允许删除端 | 等效性/备注 |
| :--- | :---: | :---: | :--- |
| **栈 (Stack)** | 一端 (Top) | 一端 (Top) | 双端队列的一端受限退化 |
| **队列 (Queue)** | 一端 (Rear) | 另一端 (Front) | / |
| **双端队列** | 两端 (L/R) | 两端 (L/R) | 最灵活，栈能做的它都能做 |
| **==输入受限==** | **一端** | 两端 | 考点核心，类似“剥洋葱” |
| **==输出受限==** | 两端 | **一端** | 考点核心，类似“拼积木” |

## 二、 输出序列合法性判决法则（功利版）

假设输入序列为 $1, 2, 3, 4$（序号递增）。

### 1. 通用公理
> 无论哪种队列，若输出序列中出现元素 $N$，意味着 $1 \sim N-1$ **必定早已进入队列**。

### 2. 三种模型的判决逻辑

#### A. 栈 (Stack)
*   **规则**：LIFO（后进先出）。
*   **秒杀口诀**：**"出大后，不能乱出小"**。
    *   即：若输出 $N$，则比 $N$ 小的元素必须按**降序**输出（除非中间穿插了更大的元素）。
    *   *反例*：`2 4 1 3`。`4`出了，栈里是`1,3`（3在顶），必须先出`3`再出`1`。`1`不能跳过`3`先出。非法。
*   **卡特兰数**：$n$ 个元素的合法出栈序列总数为 $\frac{1}{n+1}C_{2n}^{n}$。($n=4$ 时为 14 种)。

#### B. 输入受限的双端队列 (Input-Restricted)
*   **场景**：只许从**右**进，可从**左/右**出。
*   **判决逻辑（剥洋葱法）**：
    1.  观察输出序列中的最大数 $N$。
    2.  此时队列内包含 $<N$ 且未输出的所有数。
    3.  由于只能一端进，这些数在队列内的**相对顺序是固定的**（单调递增）。
    4.  **死穴**：**中间的数不能先出**。只能从两头（最小或最大）“剥离”元素。
*   **真题案例**：
    *   输入 `1 2 3 4`
    *   验证 `4 2 1 3`：
        *   `4` 先出 $\rightarrow$ 此时队内剩 `1, 2, 3`（排列必为 `[1, 2, 3]`）。
        *   要出 `2`？不行！`2` 被 `1` 和 `3` 夹在中间，两头堵死。
        *   $\therefore$ **非法**。
    *   验证 `4 2 3 1`：
        *   `4` 出 $\rightarrow$ 剩 `[1, 2, 3]`。
        *   同理 `2` 被夹住，**非法**。

#### C. 输出受限的双端队列 (Output-Restricted)
*   **场景**：可从**左/右**进，只许从**右**出。
*   **判决逻辑（拼积木法）**：
    1.  根据输出序列，反推队列内部必须存在的排列结构。
    2.  **死穴**：**相邻性破坏**。
        *   由于输入必须按 $1, 2, 3, 4$ 顺序来。
        *   $1$ 和 $2$ 无论怎么放（左插或右插），它们在队列里**必然相邻**。
        *   $3$ 只能贴在 $1,2$ 组合的左边或右边，无法插入 $1$ 和 $2$ 中间。
    3.  **核心推论**：任意相邻输入的元素（如 $k$ 和 $k+1$），在队列中不能被**后进入**的元素（$>k+1$）强行隔开。
*   **真题案例**：
    *   输入 `1 2 3 4`
    *   验证 `4 1 3 2`：
        *   要输出 `...1 3 2`（从右出），说明队内排列是 `...2 3 1 4`（右边是出口）。
        *   看 `1` 和 `2` 中间夹了个 `3`。
        *   回顾插入过程：入 `1`，入 `2`（此时 `1,2` 必相邻），再入 `3`。`3` 只能在 `[1,2]` 的左侧或右侧，**绝无可能插进 `1,2` 中间**。
        *   $\therefore$ **非法**。
    *   验证 `4 2 3 1`：同理，`2` 和 `1` 中间夹了 `3`，**非法**。

## 三、 真题速查表 (输入 1,2,3,4)

> [!danger] 重点记忆红色非法项
> 考试常考“下列哪个序列**不可能**”

| 考察类型 | 关键非法序列 (及原因) | 备注 |
| :--- | :--- | :--- |
| **栈** | `2 4 1 3` (1在3下) | 非法最多，最容易判断 |
| **输入受限** | `4 2 1 3`, `4 2 3 1` | 4出后，2被1,3夹住，无法先出 |
| **输出受限** | `4 1 3 2`, `4 2 3 1` | 1和2必然相邻，无法被3隔开 |

> [!summary] 总结：如何不丢分？
> 1.  **做题**：拿到序列，先模拟**栈**（最快排除）。
> 2.  **输入受限**：画图，先把前面的数按顺序写好（如 `1 2 3`），看能不能只抠两头把目标序列抠出来。
> 3.  **输出受限**：看相邻数（如 `1,2`）中间有没有插进**更大的数**（如 `3`）。如有，直接判错。
